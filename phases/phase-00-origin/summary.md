# Phase 00 - 项目起源 / Origin

## Goal / 目标

Define the project as a separate public brand.  
把项目定义成一个独立的公开品牌，而不是旧仓库的延伸。

## Why This Exists / 为什么要做

这个项目的核心不是“搞一篇厕刊文章”，而是：

- 用公开创作的方法把一篇 SHIT 风格论文做出来
- 把每一个阶段都保存成可分享、可复用的资产
- 把“小红书图文传播”与“GitHub 长期沉淀”接起来

This repo exists to connect public process, social shareability, and reusable creative scaffolding.

## Isolation Decisions / 隔离决策

- same GitHub account, new repo
- new brand: `VIBE SHITTING`
- no shared folder layout with the old `codex` repo
- repo tone is creative and satirical, not general experimentation

## Deliverables / 本阶段产物

- repo name and project title
- README hero and positioning
- roadmap and manifesto
- contribution rules and templates

## Done Criteria / 完成标准

- a stranger can tell in under 30 seconds what this repo is
- the repo is clearly not the old `codex`
- the first launch post can be drafted from the files in this phase

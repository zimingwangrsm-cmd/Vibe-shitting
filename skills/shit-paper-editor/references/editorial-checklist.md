## Rewrite order

1. Lock title, author line, affiliation, and keyword set.
2. Reduce the concept stack.
3. Rewrite abstract around one central claim.
4. Rewrite theory so first mention of each named term includes:
   - proposal
   - definition
   - validation logic
   - theory anchor
5. Rewrite results so the joke sits inside the evidence, not only in the framing.
6. Rewrite conclusions as outrageous but defensible claims.
7. Update glossary and review.

## SHIT-fit rubric

- Tiny scene, huge psychic cost.
- Formal method wrapped around a socially true absurdity.
- Deadpan tone.
- System critique rather than personal attack.
- At least one line that makes the reader laugh because it is too accurate.

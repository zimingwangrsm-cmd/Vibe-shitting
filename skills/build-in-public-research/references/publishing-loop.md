- `README.md`

## Minimal loop after each major change

1. Update the artifact.
2. Rebuild previews or derived assets if needed.
3. Verify the repo state.
4. Append the memo.
5. Update one public-facing markdown asset.

## Good recap ingredients

- one sentence on what changed
- one sentence on why it matters
- one sentence on what file is now canonical
- one sentence on the next likely move

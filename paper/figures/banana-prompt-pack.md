# Banana Prompt Pack / Banana 科研绘图提示词包

## Rule / 使用规则

Use Banana for narrative illustration, not for the primary numeric charts.

适合 Banana 的是叙事型科研插图，不是主数值图表。

Keep these items scripted in SVG:

- timing curves
- response-window bars
- publication U-shape
- red-envelope shock comparison

## Prompt 01 / 角色生态图

**Goal / 目标**  
Draw the lab group-chat ecosystem as a deadpan academic poster.

**Prompt / 提示词**

`A deadpan academic infographic poster showing a graduate lab group-chat ecosystem, beige and black editorial style, journal-like layout, Chinese social-science humor, labeled species including gatekeeping dog, muddy-water fish, dragging bottle, first-author junior, and procurement-reimbursement bottleneck, serious typography, flat vector illustration, no photorealism, no anime, no 3D, no watermark, no extra hands, clean white margins.`

## Prompt 02 / 点名效应三联画

**Goal / 目标**  
Explain how a message changes after an advisor directly names someone.

**Prompt / 提示词**

`A three-panel editorial diagram of an advisor group chat before and after a direct @ mention, panel 1 collective waiting, panel 2 advisor directly names one student, panel 3 reply window sharply compresses, serious academic poster style, muted beige palette, subtle Chinese interface references, clean vector shapes, labels in both Chinese and English, no brand logos, no watermark.`

## Prompt 03 / 红包效应解释卡

**Goal / 目标**  
Show why a tiny red envelope can outperform general supervision.

**Prompt / 提示词**

`A satirical scientific explainer card about the advisor red-envelope effect, tiny red envelope in a lab group chat triggering rapid reappearance of silent members, gratitude spikes, attendance-check vibe, deadpan research-poster aesthetic, cream background, black and rust typography, flat vector illustration, no photoreal faces, no watermark, no clutter.`

## Prompt 04 / 假期脑雾与恢复曲线辅助图

**Goal / 目标**  
Make a soft explanatory side plate for holiday fog.

**Prompt / 提示词**

`A subtle academic side illustration about post-holiday brain fog in graduate students, advisor calendar and chat notifications returning faster than cognitive clarity, humorous but serious, editorial infographic style, muted beige and gray color palette, clean vector composition, no photorealism, no watermark.`

## Recommended Workflow / 推荐工作流

1. Draft the exact structure in SVG first if the image carries a quantitative claim.
2. Use Banana only for the atmospheric plate or sidecar explanation.
3. Keep labels, axes, and numbers in repo-controlled SVG whenever the chart is cited in the manuscript.

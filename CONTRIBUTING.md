# Contributing / 参与说明

## What Helps Most / 最有帮助的参与方式

- Star the repo
- Vote on topics
- Suggest stronger titles or hooks
- Fix bilingual copy
- Point out places where the humor becomes low-effort

## Working Rules / 协作规则

### 1. Keep the joke rigorous / 让笑点保持严谨

Bad contribution:

- random profanity
- empty trolling
- real-person attacks

Good contribution:

- stronger framing
- tighter fake method
- better charts or table ideas
- cleaner bilingual wording

### 2. Preserve the archive / 不要破坏留档

Every phase must keep:

- a short summary
- a decision log
- a share pack
- artifacts

Do not remove process files just because the final version looks cleaner.

### 3. Respect safety boundaries / 尊重安全边界

No:

- real politics
- dangerous medical or crime instructions
- doxxing
- harassment

### 4. Prefer small, reusable improvements / 优先做小而可复用的改进

Great contributions include:

- a better issue template
- a cleaner phase template
- a stronger topic-vote rubric
- a more reusable social post format

## Good Issue Types / 推荐开的 Issue

- `Topic Vote`
- `Share Idea`
- `Translation Fix`

## Suggested Commit Style / 建议提交风格

- `docs: refine README hero`
- `phase: add topic-vote share pack`
- `social: add xhs post 03 draft`
- `scripts: add phase status checker`

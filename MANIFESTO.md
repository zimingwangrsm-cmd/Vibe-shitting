# Manifesto / 项目宣言

## One Sentence / 一句话

`VIBE SHITTING` = 用最公开的创作过程，写一篇最一本正经的荒谬论文。  
`VIBE SHITTING` = writing the most dead-serious absurd paper through the most public creative process.

## What We Believe / 我们相信什么

### 1. The joke should be rigorous / 笑点必须有方法

真正好笑的 SHIT 风格，不是粗口，不是乱骂，不是低级恶搞。  
Real SHIT-style humor is not profanity, abuse, or low-effort parody.

它的力量来自反差：  
Its force comes from contrast:

- a ridiculous premise
- a formal structure
- a method that looks serious
- a conclusion that is funny because it is half true

### 2. Build in public is part of the artwork / 公开创作本身就是作品的一部分

我们不只展示最终论文，也展示：

- 题目为什么被选中
- 方法为什么这样设计
- 哪个笑点被舍弃
- 哪个图表只是“看起来很厉害”

The archive is part of the final piece.

### 3. Every phase must leave a trace / 每个阶段都必须留痕

每个阶段都要留下这 4 类文件：

- `summary.md`
- `decision-log.md`
- `share-pack.md`
- `artifacts/`

If a phase cannot be shared, it is not finished.

### 4. The repo is bilingual by default / 这个仓库默认双语

中文负责共鸣和传播。  
English makes the work legible to a broader GitHub audience.

We do not translate everything perfectly at day one, but every major public-facing file must have both languages.

### 5. The project is brand-led, person-backed / 项目品牌主导，个人背书

项目品牌叫 `VIBE SHITTING`。  
The face behind it is `沧生`.

这不是匿名实验，也不是纯个人日记。  
It is neither anonymous roleplay nor a pure personal diary.

### 6. Safety is non-negotiable / 安全边界不可谈判

We do not touch:

- real political figures or geopolitical bait
- dangerous medical/crime guidance
- doxxing or real-person humiliation

Humor is not an excuse for real harm.

## What Success Looks Like / 什么算成功

Success is not only “submitted a paper.”  
成功不只是“投出去了一篇”。

Success means:

- the repo is readable to strangers
- the series is followable on Xiaohongshu
- the artifacts are reusable by someone else
- the finished paper still feels worth reading even if it never gets submitted

## The Promise / 承诺

We will make the process legible.  
我们会把过程讲清楚。

We will preserve the drafts, not only the polish.  
我们会保留草稿，而不只保留成品。

We will treat absurdity seriously.  
我们会认真对待荒谬。

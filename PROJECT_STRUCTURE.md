## Banana Usage / Banana 科研绘图怎么用

Use Banana for:

- concept plates
- scene reconstructions
- meme-adjacent explainer panels

Do not use Banana as the primary source of:

- numeric axis charts
- exact timing curves
- reproducible SVG figures

For that split, see `paper/figures/banana-prompt-pack.md`.
